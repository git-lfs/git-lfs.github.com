# git-lfs.github.com

Getting started minisite for Git LFS.

To preview:

1. Clone locally: `git clone https://github.com/git-lfs/git-lfs.github.com.git`
2. Run Jekyll: `script/server`
3. View in browser locally at: [localhost:4000](http://localhost:4000)
